## Data and privacy controls

- AI relationship candidates remain outside the database until the author explicitly applies selected rows.
- New concept candidates create only name-only empty nodes; no profile, tags, excerpts, or follow-up AI calls are added automatically.
- Discovery context is capped at 200 lightweight motif records and excludes article bodies by default.
- Run snapshots expose attachment type, reference ID, name, and character count, not attachment bodies.
- Release staging is scanned for key-shaped secrets, databases, private writing, logs, local test results, and private screenshots before commit and packaging.
