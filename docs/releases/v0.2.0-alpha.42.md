# Writer 0.2.0-alpha.42

This release improves article collections and the main shell layout.

## Highlights

- Collections now support selecting multiple articles at once, with clearer
  article cards and empty states.
- Articles show which collections they already belong to in the right context
  pane.
- The current article can be added to one or more collections directly from the
  right context pane.
- Collection article order can be changed by drag-and-drop, with the existing
  move buttons still available.
- The right context pane now collapses cleanly from the top toolbar instead of
  relying on splitter dragging.

## Verification

- Targeted UI and collection tests passed.
- Full `python -m pytest` passed before packaging.
- Windows portable build produced both the versioned zip and `Writer-portable.zip`.
