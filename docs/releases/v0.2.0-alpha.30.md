# Writer 0.2.0-alpha.30

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.30-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.30/Writer-0.2.0-alpha.30-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

## Highlights

- Fragment notes now live on a right-side sticky-note board instead of a rigid top list.
- Notes can be pinned, edited, marked done / restored, recolored, resized, dragged, and arranged beside the editor.
- Note board layout is stored per note, while collapsed / completed visibility is remembered per fragment during the session.
- AI now labels this feature as fragment notes; continue and expand include open notes by default.
- Legacy databases are migrated with note-board layout fields without changing existing note content.
- Delete-key typewriter scrolling was hardened so text deletion does not restart a pending follow-scroll animation.

# Writer 0.2.0-alpha.30 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.30-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.30/Writer-0.2.0-alpha.30-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

## 主要更新

- 片段便签从顶部僵硬列表改成编辑器右侧的便利贴式便签板。
- 便签支持固定、编辑、完成 / 恢复、改颜色、改宽度、拖动摆放和一键整理。
- 每张便签的位置、颜色、宽度会持久保存；每个片段的收起状态会在当前会话中记住。
- AI 工作区统一改称“片段便签”；续写和扩写默认带入未完成便签作为参考。
- 旧数据库会自动补齐便签板布局字段，不会改动已有便签内容。
- 加固删除键下的打字机滚动抑制，避免删除文字时重新触发跟随滚动。
