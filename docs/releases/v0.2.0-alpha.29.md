# Writer 0.2.0-alpha.29

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.29-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.29/Writer-0.2.0-alpha.29-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

## Highlights

- The Settings dialog can now be resized and maximized.
- Settings content scrolls when the window is small or when a provider shows extra fields.
- Save / Cancel stays fixed at the bottom instead of being pushed off-screen.
- Added regression coverage for compact Settings dialog layouts.

## Validation

- Settings / shell / startup UI tests: 80 passed.
- Full test suite: 727 passed.

# Writer 0.2.0-alpha.29 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.29-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.29/Writer-0.2.0-alpha.29-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

## 主要更新

- 设置窗口现在可以缩放和最大化。
- 窗口较小或模型提供方展开额外字段时，设置内容会进入滚动区域。
- 保存 / 取消按钮固定在底部，不再被挤出屏幕。
- 增加小窗口设置布局的回归测试。
