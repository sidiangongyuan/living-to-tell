# Writer 0.2.0-alpha.22

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.22-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.22/Writer-0.2.0-alpha.22-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

Windows may show a SmartScreen warning for unsigned alpha builds. Verify that the archive is downloaded from this repository's release assets before running it.

## Highlights

- Continuation notes no longer have an ambiguous left-side checkbox that can make a note disappear by accident.
- Each note now has explicit actions: edit, save, cancel, pin, done, and delete.
- The notes card includes a direct "Continue with notes" shortcut that opens AI Continue for the current fragment and turns note context on.
- Completion copy now makes the behavior clearer: done notes are hidden from the active continuation list; they are not part of the main prose.
- This is a UX stabilization release for the fragment continuation notes introduced in alpha.21.

## Validation

- Full test suite: 701 passed.
- Windows portable build completed successfully.
- Release artifact: `dist/Writer-0.2.0-alpha.22-portable.zip`.

# Writer 0.2.0-alpha.22 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.22-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.22/Writer-0.2.0-alpha.22-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

未签名的 alpha 版本可能触发 Windows SmartScreen 提示。运行前请确认压缩包来自本仓库 release assets。

## 主要更新

- 续写提示不再使用左侧容易误触的 checkbox，避免点一下便签就从列表里消失。
- 每条便签都有明确操作：编辑、保存、取消、置顶、完成和删除。
- 便签卡新增“用提示续写”，会直接打开 AI 续写、目标设为当前片段，并自动带入续写提示。
- 完成操作的含义更清楚：它只是从当前未完成提示列表中隐藏，不会写进正文。
- 这是 alpha.21 片段续写提示功能的体验稳定版。
