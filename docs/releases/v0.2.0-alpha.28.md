# Writer 0.2.0-alpha.28

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.28-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.28/Writer-0.2.0-alpha.28-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

## Highlights

- Fixed the brief flash of temporary label windows that could appear while starting Writer or opening the reference library.
- Fixed the same construction-order issue in the AI specimen picker.
- Stabilized reference-library layout restoration so saved splitter sizes do not trigger initialization-time feedback.
- Added regression tests that detect accidental top-level label windows.

## Validation

- Startup, reference library, and specimen picker tests: 71 passed.
- Full test suite: 726 passed.
- Windows portable build: completed (`dist/Writer-0.2.0-alpha.28-portable.zip`, 74,142,995 bytes).

# Writer 0.2.0-alpha.28 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.28-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.28/Writer-0.2.0-alpha.28-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

## 主要更新

- 修复启动 Writer 或打开文脉标本库时，临时标签窗口短暂闪现的问题。
- 同步修复 AI 添加文脉标本选择器里的同类构造顺序问题。
- 稳定文脉标本库布局恢复流程，避免恢复分栏尺寸时触发布局反馈。
- 增加回归测试，专门检测意外的顶层标签窗口闪现。
