# Writer 0.2.0-alpha.31

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.31-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.31/Writer-0.2.0-alpha.31-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

## Highlights

- Fragment notes now float over the editor surface instead of being trapped in a narrow right-side card area.
- Dragging is smoother and no longer snaps harshly on release.
- The pin control is directly on each note; pinned notes stay fixed and cannot be dragged accidentally.
- The collapsed note-board button is wider and readable, and the board can be opened / closed reliably.
- The board layout is more adaptive and avoids squeezing note text into unreadable narrow rails.

## Validation

- Full test suite: 730 passed.
- Windows portable build: completed (`dist/Writer-0.2.0-alpha.31-portable.zip`, 74,156,238 bytes).

# Writer 0.2.0-alpha.31 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.31-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.31/Writer-0.2.0-alpha.31-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

## 主要更新

- 片段便签不再被困在右侧窄栏，而是浮动在编辑器表面。
- 拖动更连续，松手时不再生硬吸附。
- 每张便签上直接提供 pin 按钮；固定后的便签不会被误拖走。
- 收起按钮变宽，文字不再被裁切，并且可以稳定展开 / 收回。
- 便签板更自适应，避免把便签文字挤到不可读。

- 全量测试：730 passed。
- Windows 便携版打包：已完成（`dist/Writer-0.2.0-alpha.31-portable.zip`，74,156,238 bytes）。
