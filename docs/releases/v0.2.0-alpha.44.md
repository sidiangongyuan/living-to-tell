# Writer 0.2.0-alpha.44

Bugfix release for the alpha.43 UI refresh.

## Fixes

- The right context pane can be freely dragged narrower again. The splitter's
  normalize pass preferred the larger of the live and remembered widths, which
  acted as a grow-only ratchet and could freeze the handle entirely.
- Calendar day cells in the Dates view stay readable across light/dark
  switches. Qt's calendar derives cell backgrounds from palette snapshots that
  lag one theme switch behind; the cells now use explicit theme-token formats
  re-applied on every theme change (this also folds the default red weekends
  into the theme's secondary text colour).
- "Manage quotes" on the daily quote card now locates the displayed quote in
  the reference library: its shelf group is selected and the passage is
  scrolled into view.

## Verification

- Full `python -m pytest` passed (759 tests, including four new regression
  tests covering all three fixes).

# Writer 0.2.0-alpha.44 中文说明

alpha.43 界面重做后的缺陷修复版本。

## 修复

- 右侧上下文栏恢复自由拖动:此前归一化逻辑取"当前宽度与记忆宽度的较大值",
  形成只增不减的棘轮,最终把拖动手柄完全卡死。
- 日期视图的日历格子在浅色/深色切换后保持可读:Qt 日历的格子背景取自调色板
  快照、滞后一次主题切换;现改为显式主题 token 格式并在每次切换时重设
  (顺带把默认的红色周末并入主题次级文字色)。
- 每日引用卡片的"管理文脉库"按钮现在会在文脉库中直接定位当前显示的句子:
  自动选中其所属书架分组并滚动到该句。

## 验证

- 完整 `python -m pytest` 通过(759 项,含覆盖三处修复的 4 个新回归测试)。
