# Writer 0.2.0-alpha.36

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.36-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.36/Writer-0.2.0-alpha.36-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

## Highlights

- The outer shell splitter now persists user-adjusted main area and right context-pane widths.
- Splitter handles are easier to grab, increased from a nearly invisible 1 px to a 7 px interaction area.
- Restored shell sizes remain guarded so the context pane cannot return at an unusable width.
- Added regression coverage for shell splitter drag persistence and handle usability.

## Validation

- Full test suite: 738 passed.

# Writer 0.2.0-alpha.36 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.36-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.36/Writer-0.2.0-alpha.36-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

## 主要更新

- 外层主布局 splitter 会保存用户调整后的主工作区和右侧信息栏宽度，重启后恢复。
- splitter 拖拽把手从几乎看不见的 1px 提升为 7px，更容易抓住和调整。
- 恢复外层布局时仍保留宽度保护，右侧信息栏不会恢复到不可用宽度。
- 增加主布局拖动持久化和把手可操作性的回归测试。

- 全量测试：738 passed。
