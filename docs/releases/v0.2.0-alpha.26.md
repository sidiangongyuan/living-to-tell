# Writer 0.2.0-alpha.26

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.26-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.26/Writer-0.2.0-alpha.26-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

## Highlights

- Continuation notes now stay collapsed when returning to a fragment unless you explicitly open them.
- Existing notes loaded during fragment switching no longer count as newly added notes, so they do not auto-expand unexpectedly.
- Note rows are denser and text-first: a small check button handles done/restore, while edit, pin, and delete are lightweight inline actions.
- Completed notes use a TODO-like visual treatment with muted struck-through text.
- The notes list grows only as needed, showing several notes at once while avoiding clipped long notes.
- Settings now include a default-collapse option for continuation notes.

## Validation

- Editor comfort tests: 38 passed.
- Targeted UI regression tests: 110 passed.
- Full test suite: 716 passed.
- Windows portable build: completed.

# Writer 0.2.0-alpha.26 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.26-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.26/Writer-0.2.0-alpha.26-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

## 主要更新

- 续写提示收起后，切换片段再回来会保持收起，除非你主动展开。
- 切换片段加载已有提示时，不再被误判为“新增提示”，因此不会自动展开。
- 提示行改成更紧凑的清单式布局：小圆形按钮负责完成/恢复，编辑、置顶、删除变成轻量行内操作。
- 已完成提示采用类似 TODO 的划线和弱化显示，仍可恢复。
- 多条提示会按数量增大可视区域，一次能看到更多提示，同时避免长提示被裁切。
- 设置页新增“默认收起续写提示”开关。
