# Writer 0.2.0-alpha.41

## Highlights

- Writer now presents **Articles** as the primary writing unit.
- Collections now contain ordered articles directly and can be exported as
  TXT, Markdown, or DOCX.
- The old visible Works / section workflow has been removed from the product
  surface, while legacy database tables remain untouched for safety.

## Changes

- Removed the Works rail entry and visible work-section AI write-back paths.
- Reworked Collections into a reading-first anthology view with article cards,
  reorder controls, removal, collection notes, and article preview.
- Added a one-time legacy import prompt that can copy old Works sections into
  articles and article collections.
- Focused global search on article results.
- Updated README, TODO, user guides, and changelog for the article /
  collection model.

## 中文

- Writer 现在以「文章」作为主写作单位。
- 作品集直接收录文章，可排序、预览，并导出 TXT / Markdown / DOCX。
- 旧「作品 / 节块」入口从产品界面移除，但旧数据库表会保留，避免误删历史数据。
- 首次检测到旧作品数据时，会提示是否复制导入为文章和文章作品集。
