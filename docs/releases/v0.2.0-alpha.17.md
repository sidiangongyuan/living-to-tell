# Writer 0.2.0-alpha.17

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.17-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.17/Writer-0.2.0-alpha.17-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

Windows may show a SmartScreen warning for unsigned alpha builds. Verify that the archive is downloaded from this repository's release assets before running it.

## Highlights

- Reopening `Writer.exe` now wakes the existing Writer process instead of starting a duplicate process.
- Soft paging now has previous / next page controls and page counts in fragment and work editors.
- The writing settings include paper vertical padding and page gap controls, alongside font, line height, paragraph spacing, and content width.
- AI tasks now show the current selected text inside the AI workspace, with copy and return-to-selection actions.
- AI can safely replace a selected range inside a work section after taking a work snapshot.

## Privacy notes

Writer stores writing data locally by default. AI providers receive only the text you explicitly send to them. This release does not include personal writing samples, local databases, API keys, OAuth tokens, account identifiers, or generated development logs.

## Validation

- Full test suite: 678 passed.
- Windows portable build completed successfully.
- Release artifact: `dist/Writer-0.2.0-alpha.17-portable.zip`.

# Writer 0.2.0-alpha.17 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.17-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.17/Writer-0.2.0-alpha.17-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

未签名的 alpha 版本可能触发 Windows SmartScreen 提示。运行前请确认压缩包来自本仓库 release assets。

## 主要更新

- 再次打开 `Writer.exe` 会唤醒已有 Writer 进程，不再启动重复进程。
- 软分页加入上一页 / 下一页和页码显示，片段与作品编辑器都可使用。
- 写作设置新增纸页上下留白和页间距，并保留字体、行距、段距和正文宽度控制。
- AI 工作区会显示当前选中的原文，支持复制和一键回到原文选区。
- 作品节块中的选区也可以被 AI 安全替换，写回前会保存作品快照。

## 隐私说明

Writer 默认把写作数据保存在本机。AI 提供方只会收到你主动发送的文本。本次发布不包含私人写作样本、本地数据库、API key、OAuth token、账号标识或开发日志。
