# Writer 0.2.0-alpha.33

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.33-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.33/Writer-0.2.0-alpha.33-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

## Highlights

- Fragment notes no longer occupy a permanent right-side layout column, so the editor and context pane keep their space on wide windows.
- The collapsed note entry is now a compact "Notes / 便签" floating pill instead of a wide text rail.
- Sticky notes are parented to the full editor panel layer, giving them a larger placement area beyond the body content wrapper.
- Added regression coverage for note-board geometry and context-pane visibility with notes expanded.

## Validation

- Full test suite: 733 passed.

# Writer 0.2.0-alpha.33 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.33-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.33/Writer-0.2.0-alpha.33-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

## 主要更新

- 片段便签不再作为固定右侧列占用布局空间，最大化后不会继续挤压编辑器和右侧信息栏。
- 收起入口改为短小的“便签”浮动胶囊，不再显示占列宽的“展开便签”。
- 便利贴挂到整个编辑面板层，可放置范围从正文容器扩大到完整编辑区域。
- 增加便签浮动布局和右侧信息栏可见性的回归测试。
