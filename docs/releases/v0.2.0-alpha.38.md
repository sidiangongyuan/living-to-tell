# Writer 0.2.0-alpha.38

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.38-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.38/Writer-0.2.0-alpha.38-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

## Highlights

- Reference-library stats now use scroll-safe expanded pages, so Chinese labels and cards are not crushed into unreadable thin rows.
- Stats cards, progress rows, tag chips, duplicate-risk cards, and recent cards now keep enough height for readable Chinese text.
- Fragment-note panel expansion now recalculates its overlay geometry immediately; no splitter drag is needed to make the panel render correctly.
- Collapsing fragment notes now hides all sticky-note windows and leaves only the compact `Notes / 便签 · count` entry visible.

## Validation

- Full test suite: 745 passed.
- Windows portable build: completed (`dist/Writer-0.2.0-alpha.38-portable.zip`, 74,164,408 bytes).

# Writer 0.2.0-alpha.38 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.38-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.38/Writer-0.2.0-alpha.38-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

## 主要更新

- 文脉标本库统计展开后改为可滚动的安全布局，中文标签和统计卡片不会再被压成细条。
- 总览、用途/类型进度条、标签、重复风险、最近条目的文字都增加了高度保护，避免中文裁切。
- 片段便签面板展开后会立即重新计算位置和尺寸，不再需要拖动 splitter 才显示正常。
- 收起片段便签时会隐藏所有便签窗，只保留紧凑的 `Notes / 便签 · 数量` 入口。
