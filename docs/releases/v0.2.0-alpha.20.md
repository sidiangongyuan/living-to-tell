# Writer 0.2.0-alpha.20

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.20-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.20/Writer-0.2.0-alpha.20-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

Windows may show a SmartScreen warning for unsigned alpha builds. Verify that the archive is downloaded from this repository's release assets before running it.

## Highlights

- AI tool results are now isolated by function, so polish, expand, continue, summarize, and diagnostic tasks restore their own result panes when switching.
- The AI tools workspace now includes clear-current-result and clear-all-results actions that keep parameters, attachments, custom presets, and version history intact.
- Writers can save custom task presets from the current style / direction input and delete those custom presets inline.
- Fragments now support manual version checkpoints for deliberate rollback points without making every autosave a historical version.
- Version History labels checkpoints clearly, while `Ctrl+S` remains a normal immediate save.

## Data recovery note

Autosave keeps the current fragment up to date, but it is not a historical timeline. Use **File → Save Version Checkpoint…** before risky edits or large AI passes when you want a deliberate rollback point.

# Writer 0.2.0-alpha.20 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.20-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.20/Writer-0.2.0-alpha.20-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

未签名的 alpha 版本可能触发 Windows SmartScreen 提示。运行前请确认压缩包来自本仓库 release assets。

## 主要更新

- AI 工具结果按功能隔离，润色、扩写、续写、摘要和诊断任务切换时会恢复各自的结果区。
- AI 工具页新增“清空当前结果”和“清空全部结果”，不会清除参数、附件、自定义预设或版本历史。
- 可以把当前风格 / 方向输入保存为自定义任务预设，并在界面中删除自定义预设。
- 片段新增手动版本 checkpoint，适合在大改、AI 写回或风险操作前保存一个可回溯节点。
- 版本历史会清晰标注 checkpoint；`Ctrl+S` 仍然只是普通立即保存，不会制造大量历史记录。

## 数据恢复说明

自动保存会保持当前片段最新，但它不是历史时间线。进行风险较高的修改前，建议使用 **文件 → 保存版本快照…** 主动保存 checkpoint。
