# Writer 0.2.0-alpha.39

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.39-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.39/Writer-0.2.0-alpha.39-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

## Highlights

- Gemini API and Gemini CLI / OAuth now wait up to 120 seconds by default, reducing premature timeouts on large writing prompts.
- Advanced users can still tune Gemini waiting time with `WRITER_GEMINI_TIMEOUT_SECONDS` or `WRITER_GEMINI_CLI_TIMEOUT_SECONDS`.
- Fragment-scoped AI chat can save selected conversation text as a fragment note; if nothing is selected, it saves the latest assistant reply.
- Public README and TODO wording now focus on features, downloads, source usage, AI setup, and the product roadmap.

# Writer 0.2.0-alpha.39 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.39-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.39/Writer-0.2.0-alpha.39-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

## 主要更新

- Gemini API 与 Gemini CLI / OAuth 默认最多等待 120 秒，降低长文本请求过早超时的概率。
- 高级用户仍可通过 `WRITER_GEMINI_TIMEOUT_SECONDS` 或 `WRITER_GEMINI_CLI_TIMEOUT_SECONDS` 调整等待时间。
- 片段作用域的 AI 对话可以把选中的对话文字保存为片段便签；未选中时保存最近一条助手回复。
- README 与 TODO 改为面向公开用户的项目介绍口径，重点展示功能、下载、源码运行、AI 设置和产品路线。
