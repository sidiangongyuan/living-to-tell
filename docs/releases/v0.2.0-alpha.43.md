# Writer 0.2.0-alpha.43

**Major UI update.** This release reskins the whole app toward a calm,
Apple-style neutral look while keeping every feature and entry point in place.

## Highlights

- Reworked the colour system to a neutral grey-white palette with a single
  system-blue accent, replacing the older warm-beige (light) and teal (dark)
  tones. Light and dark are both recalibrated.
- Tightened corner radii into a hierarchy (small controls / cards / containers)
  for a more precise, less rounded feel.
- Replaced the navigation rail's emoji glyphs with Segoe Fluent Icons line
  icons, with graceful fallback to Segoe MDL2 Assets and plain glyphs when the
  icon font is missing, so the rail never renders as empty boxes.
- Refreshed the sticky writing notes into softer, theme-aware pastels so note
  text stays readable in dark mode.
- Normalised previously hardcoded highlight colours (find matches, focus-mode
  current paragraph, AI chat roles) to theme tokens, removing leftover warm and
  teal tints after the accent change.
- Bolder keyboard focus rings and a restrained mode-switch fade, respecting the
  reduced-motion setting.
- Review hardening: the rail icon font is pinned per widget (no longer relying
  on OS glyph fallback), and AI chat history re-renders on theme switches so
  role colours never go stale.

## Verification

- Full `python -m pytest` passed (755 tests) after each phase.
- Manual light/dark walkthrough across navigation, lists, editor, AI,
  collections, dates, reference library, sticky notes, find, and focus mode.
- Windows portable build produced both the versioned zip and
  `Writer-portable.zip`.

# Writer 0.2.0-alpha.43 中文说明

**UI 重大更新。** 本次将整个应用重新换肤为安静、偏苹果风格的中性观感,同时
保留全部功能与入口。

## 亮点

- 配色系统重做为中性灰白底 + 单一系统蓝强调色,替换旧的暖米色(浅色)与
  青色(深色)。浅色 / 深色双套同时校准。
- 圆角收紧并层级化(小控件 / 卡片 / 容器),更精密、不再圆润。
- 导航栏的 emoji 字形替换为 Segoe Fluent Icons 线性图标,缺字体时优雅回退到
  Segoe MDL2 Assets 及普通字形,导航栏不会再渲染成空白方块。
- 便利贴改为更柔和、随主题适配的粉彩色,深色模式下贴纸文字保持可读。
- 把原先写死的高亮色(查找匹配、专注模式当前段、AI 聊天角色)归一到主题
  token,消除换强调色后残留的暖色与青绿。
- 更明显的键盘焦点环,以及克制的模式切换淡入微交互,遵循"减少动效"设置。
- 审查加固:导航图标字体按控件钉死(不再依赖系统字形回退),AI 聊天记录在
  主题切换时重新渲染,角色颜色不会残留旧主题。

## 验证

- 每个阶段后均跑通完整 `python -m pytest`(755 项)。
- 浅色 / 深色人工走查:导航、列表、编辑器、AI、选集、日期、参考库、便利贴、
  查找、专注模式。
- Windows 便携版打包生成版本化 zip 与 `Writer-portable.zip`。
