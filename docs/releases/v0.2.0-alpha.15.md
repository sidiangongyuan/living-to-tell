# Writer 0.2.0-alpha.15

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.15-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.15/Writer-0.2.0-alpha.15-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

Windows may show a SmartScreen warning for unsigned alpha builds. Verify that the archive is downloaded from this repository's release assets before running it.

## Highlights

- Responsive writing layout for maximized windows and focus mode.
- Dates view with a daily reference card drawn from the existing reference library.
- Quick capture window with tray-friendly workflow and configurable global hotkeys.
- Style specimen library for saving reference passages and using them as AI style constraints.
- AI Workspace for scoped writing tasks, chat, library Q&A, and safe write-back.
- Works, sections, collections, version history, search, and TXT / Markdown / DOCX export.
- Local-first data model with runtime-only credential reading.

## Privacy notes

Writer stores writing data locally by default. AI providers receive only the text you explicitly send to them. This release does not include personal writing samples, local databases, API keys, OAuth tokens, account identifiers, or generated development logs.

- Full test suite: 597 passed.

# Writer 0.2.0-alpha.15 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.15-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.15/Writer-0.2.0-alpha.15-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

未签名的 alpha 版本可能触发 Windows SmartScreen 提示。运行前请确认压缩包来自本仓库 release assets。

## 主要更新

- 最大化窗口和专注模式下，写作区域会根据窗口宽度响应式展开。
- 日期页每日文脉卡片直接从现有文脉库中抽取内容。
- 快速速记窗口支持托盘友好工作流和可配置全局快捷键。
- 文脉标本库可保存参考段落，并作为 AI 风格约束使用。
- AI 工作区支持作用域任务、对话、资料库问答和安全写回。
- 支持作品、节块、作品集、版本历史、搜索，以及 TXT / Markdown / DOCX 导出。
- 本地优先存储，凭据只在运行请求时读取。

## 隐私说明

Writer 默认把写作数据保存在本机。AI 提供方只会收到你主动发送的文本。本次发布不包含私人写作样本、本地数据库、API key、OAuth token、账号标识或开发日志。
