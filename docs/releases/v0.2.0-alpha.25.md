# Writer 0.2.0-alpha.25

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.25-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.25/Writer-0.2.0-alpha.25-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

Windows may show a SmartScreen warning for unsigned alpha builds. Verify that the archive is downloaded from this repository's release assets before running it.

## Highlights

- Continuation notes now remember their collapsed state per fragment during switching.
- Long continuation-note lists stay inside a capped internal scroll area, preserving body-editor space.
- Fragment context now exposes Save checkpoint, Versions, and Export Markdown shortcuts.
- Version History can delete the selected stored version, making old checkpoints and snapshots cleanable.
- The save model is clearer: checkpoints are stored in Writer's local version history, while exports still ask for a save path.

## Validation

- Targeted UI/service tests: 116 passed.
- Full test suite: 715 passed.
- Windows portable build completed successfully.
- Release artifact: `dist/Writer-0.2.0-alpha.25-portable.zip`.

# Writer 0.2.0-alpha.25 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.25-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.25/Writer-0.2.0-alpha.25-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

未签名的 alpha 版本可能触发 Windows SmartScreen 提示。运行前请确认压缩包来自本仓库 release assets。

## 主要更新

- 续写提示会按片段记住收起状态，切换片段再回来不会自动展开挤压正文。
- 很多续写提示会限制在卡片内部滚动，不再无限撑高编辑区。
- 片段右侧上下文面板新增“保存快照”“版本”“导出 Markdown”快捷入口。
- 版本历史支持删除所选历史版本，可以清理旧 checkpoint 和快照，且不会改动当前正文。
- 保存模型更清楚：checkpoint 存在 Writer 本地版本历史里；导出仍然会弹出保存路径选择。
