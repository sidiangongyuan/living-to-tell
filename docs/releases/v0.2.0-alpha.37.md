# Writer 0.2.0-alpha.37

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.37-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.37/Writer-0.2.0-alpha.37-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

## Highlights

- Fragment notes now open as independent Writer-owned floating sticky-note windows instead of being squeezed into the editor layout.
- Sticky-note coordinates are saved relative to the main window and can be negative, so notes can sit just outside the Writer window while still following it.
- The note entry is a compact `Notes / 便签 · count` control; expanding it shows quick add, arrange, completed-note visibility, and AI handoff actions.
- `Arrange notes` keeps pinned notes fixed, then lays active notes before completed notes in a stable sticky-note wall.
- Sticky-note windows hide when switching away from Fragments or minimizing Writer, preventing notes from leaking into Works, AI, or other pages.

# Writer 0.2.0-alpha.37 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.37-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.37/Writer-0.2.0-alpha.37-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

## 主要更新

- 片段便签改成 Writer 附属的独立悬浮便签窗，不再被编辑器列宽或右侧信息栏挤压。
- 便签位置按主窗口偏移保存，并允许负数坐标，因此可以贴到 Writer 窗口外侧，同时仍会跟随主窗口移动。
- 便签入口变成紧凑的 `Notes / 便签 · 数量` 控件；展开后只保留快速新增、整理、显示已完成和带入 AI 等操作。
- `整理便签` 会保留已固定便签不动，再按稳定顺序排列未完成便签和已完成便签。
- 切换到作品、AI、日期等页面或最小化 Writer 时，悬浮便签会自动隐藏，避免串到其他页面。
