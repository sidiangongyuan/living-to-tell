# Writer 0.2.0-alpha.21

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.21-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.21/Writer-0.2.0-alpha.21-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

Windows may show a SmartScreen warning for unsigned alpha builds. Verify that the archive is downloaded from this repository's release assets before running it.

## Highlights

- Fragments now support private continuation notes for ideas like what should happen next, unresolved beats, or reminders for the next writing session.
- Continuation notes live outside the main body, export, and full-text search, so they do not pollute the actual prose.
- The editor shows a compact top notes card with quick add, done, delete, and pin actions, plus a context-pane shortcut and open-note count.
- AI continuation and expansion tasks can include open notes as reference context by default; analysis tasks keep them opt-in.
- Notes are persisted locally per fragment and are cleaned up automatically when the fragment is deleted.

# Writer 0.2.0-alpha.21 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.21-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.21/Writer-0.2.0-alpha.21-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

未签名的 alpha 版本可能触发 Windows SmartScreen 提示。运行前请确认压缩包来自本仓库 release assets。

## 主要更新

- 片段新增私有“续写提示”，适合记录接下来怎么写、下一拍往哪里走、下次继续写时要记得什么。
- 续写提示不进入正文、不参与导出，也不污染正文全文搜索。
- 编辑器顶部新增提示卡，可快速添加、完成、删除和置顶；右侧上下文栏显示未完成数量并提供快速入口。
- AI 的续写和扩写任务默认可带入未完成提示作为参考；摘要、诊断等分析任务默认不带入。
- 便签按片段本地持久保存；删除片段时对应便签会自动清理。
