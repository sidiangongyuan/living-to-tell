# Writer 0.2.0-alpha.34

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.34-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.34/Writer-0.2.0-alpha.34-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

## Highlights

- Fragment sticky notes now live on the full main workspace layer, so they can be placed beyond the text editor rectangle.
- Notes remain scoped to the active fragment and are hidden when switching to Works, AI, or other modes.
- Window maximize / restore now refreshes note controls and drag bounds.
- Added regression coverage for cross-workspace note placement and mode-switch hiding.

# Writer 0.2.0-alpha.34 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.34-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.34/Writer-0.2.0-alpha.34-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

## 主要更新

- 片段便利贴现在挂在完整主工作区层，可放到文本编辑器矩形之外。
- 便签仍然只属于当前片段；切换到作品、AI 或其他页面时会自动隐藏，不会串到别的页面。
- 最大化 / 还原窗口时会重新计算便签入口和拖动边界。
- 增加跨工作区放置便签和切换模式隐藏的回归测试。
