# Writer 0.2.0-alpha.24

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.24-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.24/Writer-0.2.0-alpha.24-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

Windows may show a SmartScreen warning for unsigned alpha builds. Verify that the archive is downloaded from this repository's release assets before running it.

## Highlights

- Made continuation notes safer to use: marking a note done now opens the completed section so the note visibly moves instead of feeling deleted.
- Added clear state text for active, pinned, and completed continuation notes, plus a permanent-delete tooltip.
- Kept editing available directly inside each note row with save and cancel controls.
- Clarified how AI uses continuation notes with task-aware status text in the AI workspace.
- Improved the empty-note AI flow: when no notes exist, the AI panel can send you back to the fragment; after adding the first note, Writer returns to AI and enables note context.
- Added regression tests for note completion visibility, editing affordances, AI status copy, empty-state guidance, and the add-note-return-to-AI flow.

# Writer 0.2.0-alpha.24 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.24-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.24/Writer-0.2.0-alpha.24-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

未签名的 alpha 版本可能触发 Windows SmartScreen 提示。运行前请确认压缩包来自本仓库 release assets。

## 主要更新

- 续写提示的“完成”行为更安全：完成后会自动展开已完成区域，让它看起来是被归档而不是被删除。
- 每条续写提示新增明确状态文字：未完成、已置顶、已完成可恢复，并给永久删除加了提示。
- 续写提示可以直接在行内编辑，支持保存和取消。
- AI 工作区补充任务感知说明，明确当前 AI 功能是否会带入续写提示。
- AI 页没有续写提示时，可以一键回到片段添加；添加第一条后会自动回到 AI，并开启提示上下文。
- 增加回归测试，覆盖完成后可见、编辑入口、AI 状态文案、空状态引导，以及添加后返回 AI 的闭环。
