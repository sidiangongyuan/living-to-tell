# Writer 0.2.0-alpha.27

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.27-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.27/Writer-0.2.0-alpha.27-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

## Highlights

- The reference library now opens as a reading-first literary shelf: source shelves on the left, passage cards in the main reading area, and the editor hidden until needed.
- Statistics are collapsed into a compact summary by default and can be expanded without permanently stealing reading space.
- Passage cards lead with the saved sentence itself, so opening a book shows the actual excerpts instead of repeating the source title.
- Creating or editing a passage opens a right-side editor drawer with the passage content at the top.
- Shelf, stats, editor drawer, and splitter layout state persist across sessions.

# Writer 0.2.0-alpha.27 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.27-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.27/Writer-0.2.0-alpha.27-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

## 主要更新

- 文脉标本库改为阅读优先：左侧书架，中间大面积摘录卡片，编辑器默认隐藏。
- 统计默认收起为一行摘要，需要时再展开，不再长期挤占阅读区域。
- 摘录卡片先显示句子本身，点进一本书后可以直接读到不同句子，而不是反复看到作品名。
- 新建或编辑时打开右侧编辑抽屉，并把段落内容放在最上方。
- 书架、统计、编辑抽屉和分栏尺寸会记住上次状态。
