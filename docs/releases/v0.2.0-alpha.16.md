# Writer 0.2.0-alpha.16

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.16-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.16/Writer-0.2.0-alpha.16-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

Windows may show a SmartScreen warning for unsigned alpha builds. Verify that the archive is downloaded from this repository's release assets before running it.

## Highlights

- Added in-document find for fragment and work editors with match counts, highlighted results, and keyboard navigation.
- Added a calmer paper-page writing surface with soft page guides, warmer editor tones, and smoother browsing.
- Added automatic epigraph preview for literary opening quotes while keeping the underlying text plain and editable.
- Improved the literary reference library with book-style shelves, source pages, richer quote cards, duplicate review, and direct deletion access.
- Refined AI-assisted workflows with clearer excerpt location support and reference specimen selection.
- Fixed mouse-wheel scrolling in long documents so a single wheel notch no longer jumps from the start to the end.

## Privacy notes

Writer stores writing data locally by default. AI providers receive only the text you explicitly send to them. This release does not include personal writing samples, local databases, API keys, OAuth tokens, account identifiers, or generated development logs.

# Writer 0.2.0-alpha.16 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.16-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.16/Writer-0.2.0-alpha.16-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

未签名的 alpha 版本可能触发 Windows SmartScreen 提示。运行前请确认压缩包来自本仓库 release assets。

## 主要更新

- 新增片段和作品编辑器内查找，支持命中数量、全文高亮和键盘跳转。
- 新增更安静的纸页写作表面，带软分页提示、暖色纸面和更顺滑的查阅节奏。
- 新增开头题辞预览，用独立题辞卡片展示文学引文，同时正文仍保持纯文本可编辑。
- 改进文脉标本库：书架式浏览、书籍内页、精句卡片、重复复核和直接删除入口。
- 优化 AI 辅助流程，补强摘录定位和文脉标本选择体验。
- 修复长文鼠标滚轮滚动过大的问题，单次滚轮不再从开头直接跳到结尾。

## 隐私说明

Writer 默认把写作数据保存在本机。AI 提供方只会收到你主动发送的文本。本次发布不包含私人写作样本、本地数据库、API key、OAuth token、账号标识或开发日志。
