# Writer 0.2.0-alpha.18

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.18-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.18/Writer-0.2.0-alpha.18-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

Windows may show a SmartScreen warning for unsigned alpha builds. Verify that the archive is downloaded from this repository's release assets before running it.

## Highlights

- Soft page navigation now refreshes after editor content and layout changes, so next / previous page controls work reliably in long fragments.
- Page counts now use real reachable scroll positions, avoiding an extra final page where next appears available but the view cannot move.
- The editor no longer draws grey page guide lines inside the text body, avoiding overlap with wrapped text and opening epigraph previews.
- Regression coverage now includes a long `绵绵`-style fragment with an opening quote and many body paragraphs.
- Delayed typewriter-scroll callbacks now exit safely during widget teardown.

## Privacy notes

Writer stores writing data locally by default. AI providers receive only the text you explicitly send to them. This release does not include personal writing samples, local databases, API keys, OAuth tokens, account identifiers, or generated development logs.

## Validation

- Full test suite: 680 passed.
- Windows portable build completed successfully.
- Release artifact: `dist/Writer-0.2.0-alpha.18-portable.zip`.

# Writer 0.2.0-alpha.18 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.18-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.18/Writer-0.2.0-alpha.18-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

未签名的 alpha 版本可能触发 Windows SmartScreen 提示。运行前请确认压缩包来自本仓库 release assets。

## 主要更新

- 软分页控件会在正文和布局变化后刷新，长片段里的上一页 / 下一页可以可靠翻动。
- 页码现在按真实可到达的滚动位置计算，避免出现最后一页“下一页可点但画面不动”的幽灵页。
- 编辑器正文里不再绘制灰色分页线，避免压到自动识别的题记和换行后的正文。
- 新增覆盖 `绵绵` 这种开头引用 + 长正文场景的回归测试。
- 打字机滚动的延迟回调在窗口关闭或测试 teardown 时会安全退出。

## 隐私说明

Writer 默认把写作数据保存在本机。AI 提供方只会收到你主动发送的文本。本次发布不包含私人写作样本、本地数据库、API key、OAuth token、账号标识或开发日志。
