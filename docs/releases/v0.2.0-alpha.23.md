# Writer 0.2.0-alpha.23

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.23-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.23/Writer-0.2.0-alpha.23-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

Windows may show a SmartScreen warning for unsigned alpha builds. Verify that the archive is downloaded from this repository's release assets before running it.

## Highlights

- Refreshed the UI direction toward a calmer literary writing desk, with a documented design checklist for future visual QA.
- Replaced hard-coded preview colours with semantic theme surfaces so dark mode no longer leaks warm-white editor blocks.
- Split toolbar controls into clear Sidebar and Context actions instead of ambiguous arrow symbols.
- Made AI text previews and instruction fields more readable by relaxing fixed-height boxes.
- Reused no-wheel combo and spin controls across AI, settings, reference library, specimen picker, and work/include dialogs.
- Kept completed continuation notes recoverable from the editor instead of making them feel deleted.
- Reduced scrollbar visual noise for a quieter writing and reading surface.

# Writer 0.2.0-alpha.23 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.23-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.23/Writer-0.2.0-alpha.23-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

未签名的 alpha 版本可能触发 Windows SmartScreen 提示。运行前请确认压缩包来自本仓库 release assets。

## 主要更新

- 将界面方向进一步收敛为安静、克制、有文学感的写作桌面，并补充后续人工验收清单。
- 用语义化主题表面替代硬编码预览色，避免深色模式出现突兀浅色块。
- 顶部工具栏拆成明确的“侧栏”和“上下文”操作，不再使用含义漂移的箭头符号。
- 放宽 AI 选区、补充指令、粘贴文本、续写提示、附件和聊天输入的固定高度，长文本更易读。
- 在 AI、设置、文脉标本库、标本选择器、作品与包含片段对话框中复用防滚轮误改控件。
- 已完成的续写提示可以在编辑器中展开并恢复，不再像被删除。
- 降低滚动条视觉噪音，让阅读和写作表面更安静。
