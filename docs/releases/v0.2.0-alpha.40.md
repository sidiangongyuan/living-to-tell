# Writer 0.2.0-alpha.40

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.40-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.40/Writer-0.2.0-alpha.40-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

## Highlights

- Fragment notes now open from the right context pane, keeping the writing page clear when notes are closed.
- Closing fragment notes hides the note panel and sticky-note windows together; reopening the right-pane entry restores the current fragment's notes.
- The note panel is anchored to the right context pane, avoiding layout squeeze and delayed refreshes after splitter changes.
- AI Tools and AI Chat now share an "Add context" menu for adding fragments, literary specimens, and the current fragment's open notes.
- Current-fragment notes appear as one removable AI context item. Continue and Expand still include open notes by default; Chat only uses them when added manually.

# Writer 0.2.0-alpha.40 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.40-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.40/Writer-0.2.0-alpha.40-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

## 主要更新

- 片段便签入口移到右侧信息栏，便签关闭后不再占用正文页面空间。
- 关闭片段便签时，便签面板和所有贴纸一起隐藏；重新打开右侧入口后恢复当前片段便签。
- 便签面板锚定右侧信息栏，避免 splitter 挤压和需要拖动后才刷新的问题。
- AI 工具页和 AI 对话页统一使用“添加上下文”菜单，可添加片段、文脉标本和当前片段未完成便签。
- 当前片段便签在 AI 中显示为一个可移除的上下文项；续写和扩写仍默认带入便签，对话页则需要手动添加。
