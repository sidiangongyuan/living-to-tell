# Writer 0.2.0-alpha.32

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.32-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.32/Writer-0.2.0-alpha.32-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

## Highlights

- Fragment notes now use larger square sticky-note cards instead of narrow strips.
- Long note text scrolls inside the note itself.
- Notes can be placed across the full fragment workspace, including near the title, tags, and epigraph area, not only inside the body editor.
- Note width defaults and validation were raised so existing notes are not squeezed into unreadable sizes.

## Validation

- Full test suite: 731 passed.
- Windows portable build: completed (`dist/Writer-0.2.0-alpha.32-portable.zip`, 74,156,821 bytes).

# Writer 0.2.0-alpha.32 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.32-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.32/Writer-0.2.0-alpha.32-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

## 主要更新

- 片段便签改成更大的方形便利贴，不再是窄条。
- 长便签内容可以在便签内部滚动阅读。
- 便签可放在整个片段工作区，包括标题、标签、题辞和正文附近，不再局限于正文框内。
- 提高便签默认尺寸和存储校验范围，避免已有便签被压到不可读。
