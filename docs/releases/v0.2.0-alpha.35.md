# Writer 0.2.0-alpha.35

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.35-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.35/Writer-0.2.0-alpha.35-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

## Highlights

- Restored splitter sizes are now guarded so stale or extreme values cannot hide the right context pane.
- Fragment list / editor widths are normalized after restore, show, and resize, while still respecting intentional sidebar collapse.
- The outer shell keeps the visible context pane at a usable width.
- Added regression coverage for bad saved splitter sizes and focus-mode restore sizes.

## Validation

- Full test suite: 736 passed.
- Windows portable build: completed (`dist/Writer-0.2.0-alpha.35-portable.zip`, 74,159,372 bytes).

# Writer 0.2.0-alpha.35 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.35-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.35/Writer-0.2.0-alpha.35-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

## 主要更新

- 恢复 splitter 尺寸时加入保护，旧的或极端的保存值不会再把右侧信息栏挤没。
- 片段列表 / 编辑器宽度会在恢复、显示、窗口缩放时归一化，同时保留用户主动收起侧栏的行为。
- 外层主布局会保证右侧信息栏在可见时保留可用宽度。
- 增加异常保存布局和焦点模式恢复布局的回归测试。

- 全量测试：736 passed。
- Windows 便携版打包：已完成（`dist/Writer-0.2.0-alpha.35-portable.zip`，74,159,372 bytes）。
