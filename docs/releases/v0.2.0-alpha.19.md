# Writer 0.2.0-alpha.19

Writer is an alpha-stage, local-first writing desk for fragments, long-form drafts, literary references, and optional AI-assisted rewriting.

## Download

- Windows portable zip: [Writer-0.2.0-alpha.19-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.19/Writer-0.2.0-alpha.19-portable.zip)

Unzip the archive and run `Writer.exe` inside the `Writer` folder.

Windows may show a SmartScreen warning for unsigned alpha builds. Verify that the archive is downloaded from this repository's release assets before running it.

## Highlights

- AI target and output-destination controls no longer change from accidental mouse-wheel scrolling.
- Replacing a selection, a full fragment, or a work section now requires an explicit confirmation dialog.
- Fragment AI write-back now exposes an "Undo last write-back" action backed by version history.
- After AI write-back or undo, the fragment editor is synchronized with the database to prevent stale hidden editor content from overwriting the restored text.
- Regression tests cover selection-only replacement, no-wheel AI controls, and write-back undo.

## Data recovery note

AI fragment write-back already saves an `original` version before writing. If text is overwritten, open Version History or use the new undo action immediately after write-back.

## Validation

- Full test suite: 683 passed.
- Windows portable build completed successfully.
- Release artifact: `dist/Writer-0.2.0-alpha.19-portable.zip`.

# Writer 0.2.0-alpha.19 中文说明

Writer 是一个 alpha 阶段、本地优先的写作桌面，用于收集片段、整理长文、管理文脉资料，并在需要时调用 AI 辅助改写。

## 下载

- Windows 便携 zip：[Writer-0.2.0-alpha.19-portable.zip](https://github.com/sidiangongyuan/writer/releases/download/v0.2.0-alpha.19/Writer-0.2.0-alpha.19-portable.zip)

下载后解压，进入 `Writer` 文件夹，运行 `Writer.exe`。

未签名的 alpha 版本可能触发 Windows SmartScreen 提示。运行前请确认压缩包来自本仓库 release assets。

## 主要更新

- AI 的目标对象、结果落地、模型档位、强度和输出上限控件不再被鼠标滚轮误改。
- 替换选区、替换整个片段、替换作品节块前都会弹出明确确认。
- 片段 AI 写回后会显示“撤回上次写回”，可直接恢复写回前的版本历史快照。
- AI 写回或撤回后会同步刷新片段编辑器，避免隐藏编辑器里的旧内容再次覆盖数据库。
- 新增覆盖选区替换、AI 控件禁用滚轮、写回撤回的回归测试。

## 数据恢复说明

AI 片段写回前本来就会保存 `original` 版本。如果文本被覆盖，可以打开版本历史恢复；本版开始也可以在 AI 页面直接使用“撤回上次写回”。
